# OC Studio 部署指南

## 项目文件结构
```
oc-studio/
├── index.html        ← 主页面（所有 UI 在这里）
├── api/
│   └── generate.js   ← AI API 中转函数（保护你的 Key）
├── vercel.json       ← Vercel 配置文件
└── 部署指南.md       ← 本文件
```

## 部署步骤（约 10 分钟）

### 第一步：注册 GitHub + 上传代码
1. 打开 https://github.com 注册账号（已有账号跳过）
2. 点右上角 "+" → "New repository"
3. 仓库名随便写，比如 `oc-studio`，点 "Create repository"
4. 把 `oc-studio` 整个文件夹拖进去（或用 GitHub Desktop 上传）

### 第二步：注册 Vercel 并部署
1. 打开 https://vercel.com，用 GitHub 账号登录
2. 点 "Add New Project" → 选你刚刚创建的 `oc-studio` 仓库
3. 点 "Deploy"（先不设 Key，让它部署一次）

### 第三步：配置 API Key（重要！）
1. 在 Vercel 项目页面，点 "Settings" → "Environment Variables"
2. 点 "Add New"
   - Name: `MIMO_API_KEY`
   - Value: 你的新 mimo API Key（记得先去官网重新生成一个！）
3. 点 "Save"
4. 回到 "Deployments" 页面，点 "Redeploy" 让配置生效

### 第四步：访问你的网页
部署完成后，Vercel 会给你一个链接，格式类似：
`https://oc-studio-xxx.vercel.app`

手机浏览器打开这个链接，就可以用了！
可以发给朋友，也可以在手机上"添加到主屏幕"变成 App 图标。

---

## 如果你还没有 mimo 官方 API Key
1. 打开 https://api.xiaomimimo.com
2. 注册/登录
3. 创建一个新的 API Key
4. 把这个 Key 填到 Vercel 环境变量里

---

## 本地预览（可选）
如果你想在部署前先本地看效果：
1. 安装 Node.js（https://nodejs.org）
2. 安装 Vercel CLI：`npm i -g vercel`
3. 在 `oc-studio` 文件夹里运行：`vercel dev`
4. 浏览器打开 `http://localhost:3000`
